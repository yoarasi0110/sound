# Sound

一個簡單的 Python 音檔工具，可以產生指定頻率的 WAV 音訊，也可以把文字轉成中文語音 MP3。

## 功能

- 產生單一頻率或多頻率混音的 WAV 檔
- 可設定音訊秒數與音量 dBFS
- 自動加入短暫淡入淡出，降低開頭與結尾的爆音
- 支援文字轉語音，預設使用繁體中文 `zh-TW`
- 可用互動式輸入，也可用命令列參數執行

## 檔案說明

| 檔案 | 說明 |
| --- | --- |
| `main.py` | 程式進入點，負責解析參數與選擇模式 |
| `audio_generator.py` | 產生 WAV 音訊，包含頻率解析、dBFS 轉換、淡入淡出與輸出 |
| `tts_generator.py` | 使用 gTTS 將文字轉成 MP3 語音 |

## 環境需求

建議使用 Python 3.10 以上 3.14 以下版本。

需要安裝的套件：

```bash
pip install numpy scipy gTTS
```

> `gTTS` 需要連線到 Google Text-to-Speech 服務，執行 TTS 模式時需要網路連線。

## 使用方式

### 互動式執行

```bash
python main.py
```

程式會先詢問模式：

```text
1 = 音頻模式
2 = 拼音模式(TTS)
```

選擇 `1` 時，依序輸入秒數、dBFS 音量與頻率。
*程式使用 dBFS 控制數位音訊振幅，0 dBFS 是數位音訊最大值，建議不要設定太接近 0，避免播放時音量過大或失真。實際聽到的聲音大小仍會受到電腦音量、耳機、喇叭影響

範例輸入：

```text
請輸入秒數: 2
請輸入 dBFS，例如 -6: -6
請輸入頻率，可用逗號分隔，例如 800 或 800,2400: 800,2400
```

輸出檔名會類似：

```text
2s_-6db_800,2400hz.wav
```

選擇 `2` 時，輸入要轉成語音的文字，程式會輸出 MP3 檔。

範例：

```text
請輸入要轉語音的文字: 你好
```

輸出檔名會是：

```text
你好.mp3
```

### 命令列執行

產生 WAV 音檔：

```bash
python main.py --mode 1 --duration 2 --db -6 --freqs 800,2400
```

產生 TTS 語音：

```bash
python main.py --mode 2 --text "你好，這是一段測試語音"
```

## 參數說明

| 參數 | 說明 | 範例 |
| --- | --- | --- |
| `--mode` | 執行模式，目前使用 `1` 代表音頻模式，`2` 代表 TTS 模式 | `--mode 1` |
| `--duration` | WAV 音檔秒數 | `--duration 2` |
| `--db` | 音量，單位為 dBFS | `--db -6` |
| `--freqs` | 頻率列表，可用逗號分隔 | `--freqs 800,2400` |
| `--text` | 要轉成語音的文字 | `--text "你好"` |

## 輸出格式

### 音頻模式

- 取樣率：44100 Hz
- 格式：16-bit PCM WAV
- 檔名格式：`秒數s_音量db_頻率hz.wav`

### TTS 模式

- 格式：MP3
- 預設語言：繁體中文 `zh-TW`
- 預設檔名：輸入文字加上 `.mp3`

## 注意事項

- 音頻模式的秒數必須大於 `0`。
- 頻率至少需要一個，且每個頻率都必須大於 `0`。
- TTS 模式的文字不可為空。
- 若輸出檔名已存在，程式會直接覆蓋同名檔案。
